Two long-running server calls (3 seconds and 6 seconds) were started when this app was launched in order to get the data needed for the View page.

If you go to the View page before 3 seconds, you will see the default value, until the first response is received at the 3 second mark.

If you go to the View page between 3 and 6 seconds, you will only see the response value from the first call but not the second.

If you go to the View page after 6 seconds, you will only see the complete data set.

Refresh the page in your browser to restart.
